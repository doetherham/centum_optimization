# Оптимизация веб-страниц (курсовой проект)

[сайт](https://doetherham.github.io/centum_optimization/lessons.html)

## Описание

Проект представляет собой набор простых HTML-страниц, оптимизированных с точки зрения производительности и SEO (поисковой оптимизации).

Цель работы — продемонстрировать базовые методы улучшения скорости загрузки страниц и повышения их видимости в поисковых системах без использования сложных фреймворков.

## Цели проекта

### Производительность

* Уменьшение времени загрузки страниц
* Снижение размера ресурсов
* Оптимизация рендеринга браузером

### SEO (поисковая оптимизация)

* Улучшение структуры HTML для поисковых систем
* Повышение доступности контента для индексации
* Улучшение метаданных страниц

## Используемые методы оптимизации

### Производительность

* Минификация HTML, CSS и JavaScript
* Оптимизация изображений
* Использование lazy loading для изображений
* Уменьшение количества HTTP-запросов
* Удаление неиспользуемого кода

### SEO

* Использование семантических HTML-тегов (`header`, `main`, `section`, `article`, `footer`)
* Добавление мета-тегов (`title`, `description`)
* Использование alt-атрибутов для изображений
* Правильная структура заголовков (`h1–h6`)
* ЧПУ (человекопонятные URL)
